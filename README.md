# Tested TG Bot Assistent

## Как запустить проект локально (желательно с Docker, но можно и без);
Проект клонируется на локальную машину. Затем создаётся файл `config.yaml' со следующим содержимым (заменить данные на свои):
```yaml
db:
  database: db
  user: test
  password: "123"
  host: localhost
  port: "5432"
tg:
  token: 9482390:jkfdsnkcs32231k
```
Затем через терминал запускаете проект командой:
```bash
python3 main.py
```
Локализация времени привязана к linux!
